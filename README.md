A small CPM Package containing C++ objects for common embedded interfaces/devices. Tailored to STM32 MCUs but can be used generically on other MCUs.

## Current Generic Interfaces üß©

* SPI
* I2C
* Serial LED Controllers
* 8-Bit I/O Expander
* Basic IO Pins
* I2C Muliplexer
* RTT (Segger)
* PWM DAC
* STM32ADC Generic
